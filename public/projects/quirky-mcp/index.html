quirky-mcp

An MCP server that gives AI clients structured tools for working with Quirk quantum circuits.

Overview

quirky-mcp connects Model Context Protocol clients to Quirk's serialized circuit format. It focuses on a small, dependable tool surface for creating circuit JSON, checking its shape, explaining its contents, and producing links that open directly in Quirk.

Key Capabilities

  • Creates a ready-to-open Bell-pair circuit.
  • Converts Quirk circuit JSON into encoded Quirk URLs.
  • Validates the minimum circuit shape before sharing it.
  • Summarizes columns, wires, controls, and common serialized gates.
  • Passes unknown gate IDs through with warnings instead of silently rejecting them.

Technical Approach

The TypeScript stdio server uses the MCP SDK for tool registration and Zod for input contracts. It implements Quirk's public URL format directly without cloning or embedding the Quirk application, keeping the integration lightweight and easy to configure in local MCP clients.

Stack

TypeScript, Model Context Protocol SDK, Zod, Vitest, Node.js, and Quirk circuit JSON.
