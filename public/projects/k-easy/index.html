K-Easy

A browser-based Boolean simplification and Verilog generation tool.

Overview

K-Easy turns minterms and optional don't-care terms into a complete, inspectable Boolean solution. Instead of returning only a final expression, it exposes the truth table, Karnaugh map, tabulation process, selected implicants, and generated hardware description.

Key Capabilities

  • Supports Boolean functions with two to four variables.
  • Generates truth tables and Gray-code Karnaugh maps.
  • Shows Quine-McCluskey combination rounds and prime implicants.
  • Produces simplified SOP and POS expressions.
  • Generates a combinational Verilog module from the solved expression.
  • Includes built-in multiplexer and full-adder examples.

Technical Approach

A typed solver layer handles parsing, validation, minimization, grouping, and Verilog generation separately from the React interface. This keeps the algorithm testable while allowing the UI to present each intermediate step as a learning and verification aid. A Python FastAPI service supports the deployed application architecture.

Stack

React, TypeScript, Vite, Python, FastAPI, Tailwind CSS, and Verilog.
