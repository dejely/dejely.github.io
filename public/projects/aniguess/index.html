Aniguess

A fast anime comparison game built around MyAnimeList ratings and favorite counts.

Overview

Aniguess presents two anime series or characters and asks the player to choose which has the higher rating or favorite count. Correct guesses build score and streaks, while the surrounding account system keeps longer-term progress visible.

Key Capabilities

  • Series Rating, Series Faves, and Character Faves game modes.
  • Live anime and character data sourced through Jikan and MAL-oriented APIs.
  • Persistent best scores, streaks, achievements, and leaderboard entries.
  • Login, registration, guest-player support, settings, and help screens.
  • Responsive result feedback and automatic progression between rounds.

Technical Approach

The React frontend separates anime, character, game, achievement, leaderboard, and user features into focused modules. Supabase provides authentication and persisted player data, while an Express proxy and direct Jikan integrations support external anime data flows.

Stack

React, TypeScript, Vite, Supabase, Express, PostgreSQL, and the Jikan API.
