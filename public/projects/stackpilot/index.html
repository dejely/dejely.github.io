StackPilot

A native control panel for local Apache, MariaDB, PHP-FPM, and PHP project workflows.

Overview

StackPilot replaces repeated terminal commands with a compact desktop dashboard for managing a local PHP development stack. It is designed around Linux systemd workflows while also providing project-level preview servers and a Windows/XAMPP compatibility mode.

Key Capabilities

  • Polls and displays Apache, MariaDB, and PHP-FPM service status.
  • Starts, stops, and restarts services through authorized system commands.
  • Surfaces recent service logs from journalctl.
  • Runs per-project PHP preview servers on available local ports.
  • Provides distro presets, editable service units, themes, and phpMyAdmin shortcuts.
  • Builds Linux AppImage and Windows installer release artifacts.

Technical Approach

The React interface communicates with a Tauri 2 backend written in Rust. Backend commands wrap systemctl, journalctl, pkexec, filesystem opening, and managed PHP child processes. Saved configuration keeps project roots, service unit names, distro presets, and interface preferences consistent between sessions.

Stack

React, TypeScript, Vite, Tauri 2, Rust, systemd, PHP, and GitHub Actions.
