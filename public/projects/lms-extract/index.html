LMS-Extract

An authorized Moodle downloader that organizes course resources into a reusable local library.

Overview

LMS-Extract discovers courses and topics available to an authenticated Moodle account, previews planned downloads, and saves accessible files into readable course and topic folders. It is intended for personal organization and repeatable backups of materials the user is already allowed to access.

Key Capabilities

  • Lists enrolled courses and topics with flexible course-name matching.
  • Provides dry-run previews before downloading individual courses or all courses.
  • Supports academic-year filters for bulk operations.
  • Tracks completed files in a manifest and skips existing duplicates.
  • Migrates matching files from older numeric course folders.
  • Supports browser-cookie login when SSO, MFA, or JavaScript blocks normal HTTP login.

Technical Approach

A Typer CLI coordinates HTTPX sessions, BeautifulSoup parsing, local manifests, and filesystem operations. Playwright provides an explicit browser-authentication fallback without bypassing LMS permissions, account enrollment, CAPTCHA, MFA, or other access controls.

Stack

Python, Typer, HTTPX, BeautifulSoup, Playwright, Rich, python-dotenv, and pytest.
