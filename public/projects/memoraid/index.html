Memoraid

An offline-first mobile study app for flashcards, tests, notes import, and progress tracking.

Overview

Memoraid is a personal-use Quizlet-style app built for native mobile study sessions. Sets, cards, history, review state, and test results remain available locally, allowing the core experience to work without a required cloud account.

Key Capabilities

  • Create and organize study sets with descriptions, tags, and cards.
  • Study with flip, swipe, shuffle, easy/hard marking, progress, and session resume.
  • Generate multiple-choice, true/false, and written-response tests.
  • Import notes from pasted text and local text files.
  • Track recent activity, due cards, test scores, and review statistics.
  • Support light, dark, and system themes with persisted preferences.

Technical Approach

The Expo application stores decks, cards, sessions, test attempts, questions, and review statistics in a versioned SQLite schema. Feature modules isolate dashboard, import, study, and testing flows, while service interfaces reserve clean extension points for future AI generation and cloud synchronization.

Stack

Expo, React Native, TypeScript, Expo Router, SQLite, Zustand, TanStack Query, NativeWind, and Secure Store.
