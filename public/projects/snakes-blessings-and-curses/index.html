Snakes-Blessings-and-Curses

Java OOP board-game engine with extendable rules and event-based tile effects for gameplay variants.

Overview

This project focused on modeling board state, tile effects, and turn rules in a way that keeps core logic reusable for different rule sets.

What Was Built

  • Object-oriented architecture for board entities, turn flow, and events.
  • Extensible tile effect handlers for snakes, blessings, curses, and custom rules.
  • Modular game loop suited for testing and future features.

Outcome

The engine delivered clean separation of concerns so new mechanics could be added without rewriting the board model.
